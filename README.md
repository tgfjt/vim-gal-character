## Usage

to Convert Normal Chars into Gal Chars.  
`:ConvertGalChar`

Normal is:  
 `"あ", "い", "う", "え", "お", "や", "ゆ", "よ", "わ", "つ", "は", "。"`

Gal is:   
`"ぁ", "ぃ", "ぅ", "ぇ", "ぉ", "ゃ", "ゅ", "ょ", "ゎ", "っ", "ゎ", "o"`
